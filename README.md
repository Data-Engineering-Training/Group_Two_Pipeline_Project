## PROJECT DETAILS:
 
1. Generate one hundred thousand records of data such as Demographics( Name, adddress, etc), transaction activity, Customer preferences(app, website, etc), communication methods(email, SMS, etc) each for 10 companies 
 
2. Build a pipeline that would: 
   i. get the data. 
  ii. ingest it into a postgres database
 
3. Query the data by answering at least 10 questions 
 
4. Draw a diagram of the pipeline using excalidraw.io
